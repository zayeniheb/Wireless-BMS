this files contain three main folders, each serving a distinct purpose. 
The first folder, reading, includes code dedicated to reading data from the ESP32-based equipment. 
The second folder, command2, extends this functionality by enabling command transmission to the equipment, followed by restarting the data reading process. 
The third folder, current-time_voltage-time, is focused on data visualization, providing scripts to generate plots of current versus time and voltage versus time for motor analysis.
