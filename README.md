Wireless-BMS
Dans ce projet WBMS, j’ai réalisé l’acquisition des données issues de capteurs NTC, ACS712 et ADS1115 pour surveiller un pack batterie. Les données sont transmises 
via Wi-Fi avec ESP32. Schéma de câblage conçu, puis adaptation d’une carte BMS TI optimisée sans blocs non essentiels pour optimiser le système pour optimisation
du système
